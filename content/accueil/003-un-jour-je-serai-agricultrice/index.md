# Un jour, je serai agricultrice...

Moi, c’est Adeline MONNERET. Si un jour on m’avait dit que je serai éleveuse de
chèvres, je n’y aurais pas cru ! Non issue du milieu agricole, une vraie NIMA
comme on dit dans le milieu, je vais vous raconter comment je suis passée de
chargée de mission développement économique dans des collectivités territoriales
à agricultrice.
